## Select multiple lines by click
> <kbd>Ctrl + Click</kbd>

## Select multiple lines by selection
> <kbd>Ctrl + Shift + Alt + Right</kbd>

## Select multiple occurrences
> <kbd>Ctrl + D</kbd>

## Comment by selection
> <kbd>Shift + Alt + A</kbd>

## Duplicate line
> <kbd>Shift + Alt + Down</kbd>

## Move line
> <kbd>Alt + Down</kbd>
